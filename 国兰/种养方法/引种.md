# 引种

由于国兰市场是一个有很强投机属性的畸形市场,里面鱼龙混杂,兰农,爱好者,有投机目的的个人养殖户,专业兰商,二道贩子都在其中有各自的生态位,卖家往往会利用信息差,专业手段忽悠买家高价买花,而兰花作为一个观赏植物,是个活物,成活才是最终目的,国兰的引种有一定的讲究.

引种的关键是如下几点:

1. 品种对版
2. 价钱合理
3. 健康没有暗伤
4. 时机合适

要做到以上我们需要从几个方面来挑选引种渠道:

1. 渠道类型
2. 植株状态
3. 发货地区

## 理清需求

在引种之前,我们也需要先理清自己的需求:

1. 有没有对应种养条件
2. 看中的特质

种植条件大致可以分为如下要素

### 环境气候要素

我们可以在[亚洲气候区域地图](https://www.plantmaps.com/koppen-climate-classification-map-asia.php)确认自己所在地区的气候类型,在[中国植物耐寒区地图](https://www.plantmaps.com/interactive-china-plant-hardiness-zone-map-celsius.php)中查看自己所在地区的全年最低温度范围.

除此之外还要考虑两个修正条件:

+ 如果你的种养位置在山上,通常每升高100米温度下降0.6摄氏度左右.
+ 秦岭淮河线以北冬季会集中供暖,室内温度恒定在25度左右且湿度极低

不同兰种可以成活,可以开花的温度范围是不同的,

+ 春兰: 成活温度(0~40)摄氏度,湿度越高状态越好;孕育花苞需要30摄氏度以上的环境;开花需要在0~15摄氏度范围内30天左右进行所谓"春化"(品种不同温度范围和持续时间也有所不同)
+ 蕙兰: 成活温度(-5~40)摄氏度,湿度越高状态越好;孕育花苞需要30摄氏度以上的环境;开花需要在0~15摄氏度范围内30天左右进行所谓"春化"(品种不同温度范围和持续时间也有所不同)
+ 莲瓣: 成活温度(0~35)摄氏度,湿度越高状态越好;孕育花苞需要25摄氏度以上的环境;开花需要在5~15摄氏度范围内10天左右进行所谓"春化"(品种不同温度范围和持续时间也有所不同)
+ 春剑: 成活温度(0~40)摄氏度,湿度越高状态越好;孕育花苞需要30摄氏度以上的环境;开花需要在0~15摄氏度范围内10天左右进行所谓"春化"(品种不同温度范围和持续时间也有所不同)
+ 建兰: 成活温度(5~40)摄氏度,湿度越高状态越好;孕育花苞需要28摄氏度以上的环境;开花需要在28~40摄氏度范围内30天左右"积温"(品种不同温度范围和持续时间也有所不同)
+ 墨兰: 成活温度(5~40)摄氏度,湿度越高状态越好;孕育花苞需要20摄氏度以上的环境;开花需要在5~10摄氏度范围内7天左右"春化"(品种不同温度范围和持续时间也有所不同)
+ 寒兰: 成活温度(5~40)摄氏度,湿度越高状态越好;

无论哪种兰花,环境湿度都是越高长得越好,昼夜温差大也会更利于长壮,而且如果环境湿度过干可能引起花苞消苞.

根据上面的信息,我们大致就可以确定自己所在地区适合养什么品种的国兰.

+ 东北: 适合建兰墨兰;春兰蕙兰莲瓣春剑需要合理利用室外低温进行春化,主要需要解决冬季湿度问题
+ 西北: 适合建兰墨兰;春兰蕙兰莲瓣春剑需要合理利用室外低温进行春化,主要需要解决全年湿度问题,冬季外昼夜温差大是优势
+ 华北: 适合建兰墨兰;春兰蕙兰莲瓣春剑需要合理利用室外低温进行春化,主要需要解决全年湿度问题
+ 长江流域: 适合春兰蕙兰莲瓣春剑;建兰墨兰需要注意冬季保温.四季分明是其优势
+ 岭南: 适合建兰墨兰;春兰蕙兰莲瓣春剑需要挑春化要求低的品种,自然开花基本靠寒潮,全年高温时间长,湿度高是优势
+ 云贵: 适合春兰蕙兰墨兰莲瓣春剑;建兰可以长得很好但开花少.四季如春昼夜温差大是其优势

总结就是全中国基本都可以养大部分国兰,但春化是影响除建兰墨兰外所有国兰品种开花的一个因素,因此岭南比较吃亏,要春化的品种很难开花,北方由于室外温度太低室内温度又太高所以春化也成问题.

在光照方面,国兰普遍喜欢明亮散射光,春秋冬的低角度斜射阳光也有利于兰花生长,但夏天的高角度直射光对多数国兰来说是难以承受的.

中国处于北部北回归线以北北纬53度34分~3度51分的区间,随着纬度升高,相同时刻阳光的照射角度会更低,光线强度也就越低.也就是说纬度越低的地方能让国兰直接晒太阳的时间越少.

水质方面,北方水碱大,长期用容易造成土壤盐碱化需要注意,解决方法有如下几种:

1. 用烧开的水冷却后浇,水烧开后钙镁离子会沉淀称为水垢,这个方法缺点就是费电费水壶
2. 接雨水浇水,这个方法缺点就是不可控,而且也占地方.
3. 平时多用矾肥水这类偏酸性的肥料施肥调节土壤酸碱度
4. 勤翻盆换土

通常上面几种方式也是结合使用的,而且即便是南方,水质也没有好到完全没有水碱,只是程度轻些而已.3,4两点也还是要做的.

### 国兰特质总结

兰花终究是观赏植物,我们养它终究是要欣赏的,在挑选时我们也需要结合自己的需求选择合适特质的品种.我总结大概有如下几种特质

> 按种养特性

+ `好养皮实`.通常来说好养程度建兰>墨兰>蕙兰>春兰>春剑>莲瓣,不同商品种又会有一些差别.追求这种特质的一般是新手或者是不想花太多精力养护的玩家.
+ `开花机器`,也就是花性好,确实有的品种非常容易开花,也有些就是懒花.追求这种特质的一般主要是为了赏花闻香的玩家
  
> 按欣赏角度

+ `花叶共赏`,通常指的是那种带叶艺且叶不容易焦尖长黑斑退草的品种,有叶艺确实可以在没花的时候增加兰花的可观赏性
+ `花品一流`,通常指的是开花周正,花品稳定性,花守一流,花叶协调的品种.能称得上花品一流的品种实话说不多见,很多花往往不能样样出色,所以能符合这一特质的花可能并不是最出彩的,但一定是你愿意一直养下去的.
+ `飘逸灵动`,也就是叶姿相对直立飘逸,筋骨佳,花品周正,花守一流.叶子太短,筋骨不佳,看起来软塌塌松松垮垮的自然飘逸不起来,所以一般不会是小株型,软叶的品种.这种特质的品种颇有文人气质,由于占用上部空间较多对阳台会不太友好.
+ `雍容华贵`,也就是叶姿相对垂,叶质厚有筋骨,叶面如果有蛤蟆皮或行龙就更好了,花端正气派,花守一流,这种特质在90,00年代比较流行,确实看起来比较贵气,有官像,但缺点是相同株型会更占地方,需要有更大的面积来配它.
+ `婀娜多姿`,也就是叶扭转卷曲弓垂,姿态多变,花型柔和饱满,花守一流,这种特质偏向柔美,仿佛大家闺秀,多是中小株型品种

> 按株型和摆放位置

+ `精致小品`,一般株型小巧,通常都是一些小叶材品种,将其种在小盆中可以放在窗边案头,可以随时拿起来欣赏把玩,关键是不太占地方.
+ `花架盆栽`,一般是中小叶材,中型叶材的品种,叶材不会超过50cm,由于株型较大需要有比较大的种养空间,通常要么种在专门种植物的阳台,露台阳光房里,要么养在光线通风很好的大客厅中.
+ `花园点缀`,一般是中型叶材到大型叶材,叶材一般都在30cm以上,长的可以到1m,一般是养在院子里
+ `盆景素材`,一般和竹,石搭配做盆景,竹高,兰瘦,石怪,相映成趣.一般用的都是细叶飘逸品种,但不会用太高大的品种.这种盆景通常用马槽盆,斗笠盆这种浅盆,要求选的兰品种根系不能太发达.

## 引种的时机

引种一般都会有一个1到2周的服盆期,新入的兰草需要适应当前的环境.服盆期中兰草除了开花外基本是停止生长的状态.引种最好在气温15摄氏度~25摄氏度之间的时候进行,这个温度下兰花生命力最强,因此也服盆最快,因此最佳时机有两个:

+ 春季,最优时机,万物竞发勃勃生机,几乎所有的植物都可以在这个时候购买.这个时期也是分苗翻盆操作的时机,因此会有不少玩家出草,选择比较多.可以分为如下几个细分时间段:
    + 清明前后,人说清明前后,种瓜种豆,这个时期春兰春剑莲瓣基本下花了开始发春芽,蕙兰正含苞待放,建兰开始发春芽,因此最合适的是引种蕙兰,如果能确保品种其他兰花品种也都是这个时期引种最易成活.这个时期温度不太稳定,需要注意倒春寒的问题.
    + 劳动节前后,春末夏初,气温基本稳定了,一些早花品种的建兰可能已经出了花芽.而蕙兰应该也差不多下花了.这个时期最适合引种建兰.

    春季引种的兰花只要上盆不马虎一般都没什么问题,但需要注意,春季温度升高,湿度也大,需要注意防虫防菌.

+ 秋季,收获的季节,气温合适,但日间温差大,湿度相对低.本来是兰花养壮的时机,这个时期通常是下了花的建兰以及没上花苞的春蕙莲瓣春剑分苗翻盆的时机,也会有不少玩家出草,选择比较多.可以分为如下几个细分时间段:
    + 中秋国庆前后,这个时期日间温度可能还在30摄氏度上下徘徊,建兰还有晚花,春兰蕙兰莲瓣春剑已经出了花苞,有春化条件的地方适合引种春兰蕙兰莲瓣春剑以及建兰,没有春化条件的地方适合引种建兰
    + 深秋初冬,11月到12月这个时间段温度已经显著下来了,建兰晚花也应该结束了,墨兰基本出花苞了,春蕙莲瓣春剑也应该春化差不过结束了.这个时机最适合引种墨兰,以及没有春化条件的地方的土豪引种春蕙莲瓣春剑当年宵花.

    秋季引种通常不太需要关心病虫害的问题,但由于接下来就是普遍休眠的冬季,中途引种服盆期会影响兰草的能量积蓄和秋芽质量.

并不是说冬季夏季不能引种,只是相对有一点风险,冬季兰花休眠,如果处理得当,可以让服盆期不影响春季生长,理论上实际是最佳引种时间,但这对经验要求比较高,不推荐;夏季高温高湿,是两腐高发时期,也是建兰的主要花期,本来引种建兰是完全可以的,但为了避免不必要的扯皮,建议等到秋天再说

## 引种的渠道

通常渠道可以分为:

+ 兰场
    + `大棚草`,天然棚兰场,也就是露天搭个棚子,像种庄稼那样种出来的兰
    + `加温草`,控温棚兰场,也就是类似温室大棚,自动喷淋浇水施肥的兰场种出来的兰
    + `激素草`,使用大水大肥,滥用各种药物激素的控温棚兰场种出来的兰
    + `瓶苗`,组培或杂交未出瓶在恒温室种养或刚出瓶未脱毒的兰
    + `硬化苗`,组培货杂交已经脱毒或已经硬化一段时间的苗,一般分硬化一年和硬化三年的苗
+ `阳台草`,阳台玩家分的苗,通常是精心照顾且经历阳台恶劣环境的兰草
+ `洗澡草`,二道贩子从一处买入后直接卖出或短暂种养后卖出的草

发货位置可以大致分为:

+ 福建两广,高温高湿,天然温室,气候导致这边发苗多,因此兰场多在此处,鱼龙混杂,有天然棚,也有控温棚,但一般还是天然棚加水帘加喷淋系统的居多,毕竟不需要加温激素这些也可以养的卖相很好,而且一般相同品种福建两广的都是最便宜的.这边的植料多是石子加树皮,通常这边的兰场的建兰墨兰春兰,只要价钱不超过单苗100的品种还是靠谱的.这边的玩家也多玩建兰墨兰.偶尔有种春蕙的一般都是资深玩家,有人为了春化花钱买冷柜,把花搬山上,寄到别的省去,相当硬核.
+ 云贵,日间温差大,四季如春,通常是天然棚为主,但一般冬季建兰还是会加温,发苗比不过福建广东因此兰场方面不太成规模.玩家不少,但多玩春蕙和本地品种.
+ 江浙,传统养兰地区,也是兰文化的主要发源地,兰场一般都是大棚草或者加温草,通常走精品路线,也确实不难养,喜欢号称自己的是老种,相同品种比福建两广能贵上几倍(老种这个话题后面讲).江浙玩家一般玩春蕙居多,当然了那边气候得天独厚也最适合养春蕙
+ 川渝,有点类似云贵到江浙的过渡状态,加温草激素草比较普遍,而且出来普遍不如江浙的好养,玩家主要是养春兰建兰春剑莲瓣为主.
+ 京津冀山东,老实说气候并不适合大范围种植,兰场几乎必然是控温棚,但兰草普遍并不算难养,主要是建兰,春蕙为主,玩家也挺多,北京天津的兰友也是一个挺大的群体.
+ 其他地区,兰场当然也有,但不太多,玩家还是有的,哪里有人出花都不奇怪

一般来说选购的时候以如下优先级挑选不容易有问题

`阳台草`>`大棚草`>`硬化三年苗`>`硬化一年苗`>`加温草`>`瓶苗`>`激素草`>`洗澡草`.

当然了卖洗澡草的肯定不会和你说他卖的是洗澡草,卖激素草的肯定也不会和你说他卖的是激素草,因此卖家可信才是最主要的因素.怎样增加可信度呢?可以从如下几个方面入手:

1. 找本地兰友或兰场.跑得了和尚跑不了庙,买之前直接去现场看,现场挑,看他真实的种养环境,一手交钱一手交货比什么都稳
2. 网购的话看店铺评价和开店时间,评价不能局限于购买平台,也可以去贴吧,中交等论坛交叉验证.一般网购就是淘宝,拼多多,抖音,中交.不建议直播间买,托比较多.当然也可以看看卖家说什么,满嘴升值,捡漏,吹的天花乱坠的最好别买.
3. 如果是走咸鱼,最好确认卖家是真的玩家而非二道贩子,可以看卖家的过往售出记录,以及过去的发帖情况和评价,咸鱼交易凡事让你不在平台上付钱的都是骗子.如果过分活跃可能是做这个生意的;如果周期性的活跃,且活跃的时候都是分苗的时间和花期,那很可能是真玩家;如果除了花外也有别的卖,或者也卖过其他二手,可能就真的是玩家;也可以和卖家聊聊,探探虚实.
4. 如果是贴吧购买,可以看看这个卖家过往的帖子,如果经常卖的一般问题不大,因为贴吧是小圈子,有问题的早就被请走了.当然了更多的是沉默的大多数,尽量要求付款走咸鱼吧,争取自己的利益.

## 苗情甄别

买回来的兰花对不对版,养不养得活,最核心的还是要做苗情甄别,甄别的点主要是:

1. 带花买(至少是带花苞),这样可以最大限度的保证对版,而且能开花的植株也瘦弱不到哪里去
2. 看芦头,首先是有几苗,无论是什么品种,肯定是苗数越多越容易成活,但苗数多,根多,上盆就不太容易,推荐3到4苗引种;然后是芦头大小,不同兰种芦头大小不同,但壮苗都有一个范围,超过范围或者不到范围的都不是好现象,比如建兰通常芦头在食指指节那么∂大,有的卖你和黄豆大小,有的卖你和独头蒜大小,这都是有问题的.
3. 看根,不同品种的兰根粗细不尽相投,还是一样,过粗或者过细都不是健康的表现;兰有没有活力主要就看水晶头,也就是根尖位置是否呈透明半透明质,水晶头越多说明越健康,如果一个没有那可能就不太行了;再有就是根的颜色,如果根发黑,很可能是肥伤,就不是健康的根了.
4. 看叶子,兰场出来的兰叶子多墨绿,偏长,几乎没有焦尖,这是阴养湿养的表现.如果你是阳台种养,环境变化过大,这种兰需要更长的适应时间;阳台草,大棚草往往都是阳养,即便有遮阳也会自然矮化,叶子会偏黄,变短,难免会有焦尖,这样的草往往抗性更好,适应能力也更强;另外就是看叶夹,干枯的叶夹伤如果有蓝色绿色粉状物,多是药物残留,长有青苔,多说明养护环境非常湿,这种都会增加适应新环境的难度.
