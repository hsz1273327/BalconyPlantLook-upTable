# 大明胭脂

+ 品种: 建兰
+ 产区: 未知
+ 来源类型: 未知
+ 价格区间: 160元/苗
+ 草型: 小叶材
+ 优点:
    + 花色优秀
    + 叶型出色
    + 极易出架

+ 缺点:
    + 花型一般
    + 较贵

## 来源

大明胭脂算是一个新品,2018年中山兰展上的得奖品种.第一次进入大众视野是在2016年,中交上深圳网友`兰色的海`发布的帖子`大明胭脂开`中.这个品种应该在该帖子之前在兰花小圈子中已经有过一定范围的传播,只是并没有找到出处.值得一提的是似乎该贴中的开品就是这个品种的最佳开品.

## 评价

+ 推荐等级: 6
+ 推荐人群: 阳台养兰爱好者,喜欢小草大花的爱好者,将兰花当案头花的爱好者,色花爱好者

建兰小叶材色花的代表,也几乎是唯一的选择(另一个选择是正红旗,不过那个叶材稍大些).一般喜欢小叶材兰花的玩家都会入.

大明胭脂虽然叫胭脂,但归类应该算紫红花,而且它的上色不太稳定,光照好的条件下上色较好是紫红色,光线不好的条件下花瓣中间位置会是琥珀色向外扩散.虽然上色不稳定,但好在两种颜色都挺好看,属于开品上限高下限也不低的品种.因此也算对的其它的价格.

## 欣赏价值

大明胭脂欣赏价值主要可以总结为小叶材红花,花大出架.建兰的小叶材花可以选的不多.
